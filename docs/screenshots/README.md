# Visual Assets / 图片资源说明

This directory stores visual assets used by the open-source documentation.

本目录用于存放开源文档中使用的图片资源。

Current strategy:

- AI-generated visuals are used for the first open-source packaging pass
- they are intended as explanatory illustrations
- maintainers can replace them later with real screenshots if desired

当前策略：

- 首轮开源包装使用 AI 生成图片
- 这些图片主要用于帮助理解结构与使用方式
- 后续维护者可以按需替换成真实运行截图

Recommended filenames:

- `hero-cover.png`
- `architecture-overview.png`
- `workflow-diagram.png`
- `ai-review-summary-demo.png`
- `ai-inline-comment-demo.png`
